## Firebase Setup for Blog

This project expects Firebase credentials via environment variables as used in `src/firebase/firebase.config.ts`. Create a `.env.local` file with the following keys from your Firebase project:

```
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
```

Enable **Firestore** in your Firebase console and create a collection named `posts`. Each document should contain:

- `title` – post title
- `date` – publication date (string)
- `slug` – unique slug used in URLs
- `content` – HTML or markdown string for the post body
- `imageUrl` – URL for the header image

After the environment variables are configured you can run `npm run dev` to start the blog locally.
